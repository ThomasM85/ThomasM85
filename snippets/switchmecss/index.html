How to use

  1. Add /css/switchme.css to your document.

    HTML

                  
                <link rel="stylesheet" href="./your/path/to/switchme.css" media="all" />
  2. Or, include it your global CSS/SASS bundle file to minimize HTTP requests.
  3. Add /js/switchme.js to your document.

    HTML

                  
                <script src="./your/path/to/switchme.js" />
  4. Or, include it your global Javascript bundle file to minimize HTTP requests.
  5. Copy/paste the snippet you want in your document / component and replace demo content with your own (how customize ?).
  6. Customize markup, CSS properties and animations to fit with your UI.
  7. That's all. Enjoy !
