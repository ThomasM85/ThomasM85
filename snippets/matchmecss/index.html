How to use

  1. Add /css/matchme.css to your document.

    HTML

                  
                <link rel="stylesheet" href="./your/path/to/matchme.css" media="all" />
  2. Or, include it your global CSS/SASS bundle file to minimize HTTP requests.
  3. Copy/paste the snippet you want in your document / component and replace demo content with your own (how customize ?).
  4. Customize markup, CSS properties and animations to fit with your UI.
  5. That's all. Enjoy !
